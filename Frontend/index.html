VoiceFlow Assistant

A local-first delivery-assistant chat widget with voice input and multilingual replies.

What you can ask

  • Delivery types, items available, delivery hours
  • Order tracking and payment methods
  • Out-of-scope questions get a guided “what to ask” reply

How to use

  1. Start the backend: npm start
  2. Open the chat (bottom-right)
  3. Pick a suggestion chip or type your question

Notes

  • Text answers work offline (from Postgres)
  • Mic transcription needs internet + HF_TOKEN
  • Weather needs internet (Open‑Meteo)

How it works

Your message is shown instantly, then the backend returns the best match from the database (intent + fuzzy + semantic). Voice uses Whisper transcription first.
